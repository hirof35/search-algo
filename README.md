
#線形探索アルゴリズム・シミュレーター (Ruby/Tk)
RubyのGUIツールキット「Ruby/Tk」を使用した、各種線形探索（リニアサーチ）アルゴリズムの挙動を視覚的に比較・学習できるデスクトップアプリケーションです。

探索のプロセスが1ステップずつ色付きでアニメーション表示されるため、各アルゴリズムの内部的な動きや仕組みの違いを直感的に理解することができます。

🚀 搭載されているアルゴリズム
本アプリでは、以下の3種類の線形探索を切り替えてテストできます。

1. 通常の線形探索 (Normal Linear Search)
配列の先頭から末尾に向かって、愚直に1つずつ要素を比較していく最もシンプルな探索方法です。

見どころ: 一致しなかった要素が次々とグレーアウトし、見つかった瞬間に緑色に変化する基本の動きが確認できます。

2. 番兵付き線形探索 (Sentinel Linear Search)
配列の末尾に「探したい値（番兵）」をあらかじめ挿入しておく手法です。これにより、ループ毎に「配列の終端に達したか？」という判定を省略できるため、内部的な処理速度が向上します。

見どころ: 実行時に右端へ青い「番兵ラベル」が出現します。データ内に値が存在しない場合、最後の番兵にヒットして探索が終了する（番兵の役割を終える）様子が視覚的にわかります。

3. 自己組織化探索：先頭移動法 (Self-Organizing Search: Move-to-Front)
「一度探索されたデータは、近い将来に再度探索される可能性が高い」という仮説（時間の局所性）に基づいた応用アルゴリズムです。

見どころ: 目的の値が見つかった瞬間、その要素が配列の先頭（一番左）へと自動的に移動します。同じ値を続けて検索すると、次回からはわずか1回の比較で見つかるようになる実用的な挙動を体感できます。

🛠 動作環境・必要な準備
Prerequisites
Ruby 2.4 以降

Tcl/Tk (OSのGUIライブラリ)

準備手順
OSに Tcl/Tk をインストール
環境に応じて、事前に Tcl/Tk をインストールしておく必要があります。

Ubuntu/Debian: sudo apt install tcl-dev tk-dev

macOS (Homebrew): brew install tcl-tk

Windows: RubyInstaller等を利用している場合、標準で同梱されていることがあります。

tk 拡張ライブラリ（Gem）のインストール
Ruby 2.4以降では Tk が標準添付から外れているため、以下のコマンドでインストールします。

gem install tk

## 💻 使い方

1. 本リポジトリのソースコードを `linear_search_app.rb` という名前で保存します。
2. ターミナル（またはコマンドプロンプト）を開き、以下のコマンドでアプリを起動します。
   ```bash
   ruby linear_search_app.rb
画面が起動したら、以下の手順で操作します。

アルゴリズムの選択: 試したい探索方法のラジオボタンにチェックを入れます。

数値の入力: 中央のテキストボックスに、探したい数値を半角数字で入力します（例: 56 や 11 など）。

探索開始: ボタンを押すと、0.4秒間隔のステップアニメーションで探索が始まります。

🎨 画面のカラーインジケータ
🟨 黄色: 現在チェックしている（比較中）要素

🟩 緑色: 目的の数値が一致した（発見された）要素

⬜ 灰色: チェックした結果、一致しなかった要素

🟦 水色: 番兵付き探索で一時的に生成された「番兵」
